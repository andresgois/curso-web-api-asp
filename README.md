# CSHAP

## Comandos Base
```
dotnet --version

dotnet new

dotnet new --list

dotnet new webapi -o demoapi2 --framework net6.0

dotnet run

dotnet --list-sdks

dotnet new console --framework net6.0
```
- [Baixar .NET 6.0](https://dotnet.microsoft.com/pt-br/download/dotnet/6.0)
